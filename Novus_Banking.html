Project Overview

Business Challenge & Solution Approach

Novus Banking required a comprehensive Salesforce solution to modernize their Fixed Deposit application process and streamline investment tracking workflows. As the lead Salesforce developer, I architected and implemented custom Lightning Web Components to support FD Details and Applicant Details objects, while establishing a robust Interest Scheme master object to manage complex investment products. The project focused on eliminating manual data entry processes, reducing operational errors, and providing real-time visibility into customer investment portfolios.

Technical Implementation & Project Devlierables

Custom Lightning Web Components

Developed sophisticated LWC components for FD Details and Applicant Details objects, providing intuitive user interfaces for investment advisors to manage customer fixed deposit applications with real-time data validation and dynamic field updates.

Interest Scheme Master Object

Architected a comprehensive Interest Scheme master object to manage over 5,000 static records, enabling dynamic interest rate calculations, term-based product configurations, and automated maturity date calculations for diverse investment products.

Investment Tracking Workflows

Streamlined investment tracking workflows through automated process flows, providing real-time visibility into customer investment portfolios, maturity schedules, and renewal opportunities for relationship managers.

Process Automation

Implemented comprehensive automation for the Fixed Deposit application lifecycle, including automatic document generation, approval routing, interest calculations, and maturity notifications, eliminating manual intervention points.

Data Integrity & Validation

Built robust validation frameworks ensuring data accuracy across all FD applications, with real-time field validation, duplicate detection, and regulatory compliance checks integrated into the user experience.

Reporting & Analytics

Created comprehensive dashboards and reports for investment performance tracking, portfolio analysis, and regulatory reporting, providing stakeholders with actionable insights into Fixed Deposit business metrics.

fetchSchemeHandler(){
    let isValid=true;
    let inputFields=this.template.querySelectorAll('.clsFetchScheme');
    console.log(inputFields);
    inputFields.forEach(e=>{
        if(!e.checkValidity()){
            e.checkValidity();
            e.reportValidity();
            isValid=false;
        }
    })
    if(isValid){
        fetchIntScheme({
            depType: this.depositTypeValue, 
            tenMonths: this.tenorInMonthsValue, 
            tenDays: this.tenorInDaysValue, 
            fdAmount: this.fdAmountValue,
            cusType: this.customerTypeValue
        })
            .then(result => {
               console.log(result);
               let options=[];
               result.forEach(e=>{
                let temObj={};
                temObj.label=e.Name;
                temObj.value=e.Id;
                temObj.intRate=e.Interest_Rate__c; 
                options.push(temObj);
               })
               this.schemeOptions=options;
            })
            .catch(error => {
                console.log(error);
            }); 
    }
} 
schemeHandler(event){
    let sch=event.target.value;
    this.schemeValue=event.target.value;
    this.schemeOptions.forEach(scheme=>{
        if(scheme.value==sch){
            this.schemeValue=scheme.value;
            this.interestRateValue=scheme.intRate;
        }
    })
} 
saveHandler(){
    let isValid=true;
    let inputFields=this.templatequerySelectorAll('.clsFetchScheme');
    inputFields.forEach(e=>{
        if(!e.checkValidity()){
            e.checkValidity();
            e.reportValidity();
            isValid=false;
        }
    })
    isValid=true;
    inputFields=this.template.querySelectorAll('.clsSave');
    inputFields.forEach(e=>{
if(!e.checkValidity()){
    e.reportValidity();
    isValid=false;
}
    })
    if(isValid){ 
        updateFd({
            fdId:this.recordId,
            cusType:this.customerTypeValue,
            depType:this.depositTypeValue,
            tenMonths:this.tenorInMonthsValue,
            tenDays:this.tenorInDaysValue,
            payFreq:this.payoutFreqValue,
            fdAmount:this.fdAmountValue,
            insSchemeId:this.schemeValue,
            intRate:this.interestRateValue
        }) 
            .then(result => {
                console.log(result);
                this.dispatchEvent(new ShowToastEvent({
                    title: "Success!",
                    message: "Successfully Record is Updated!",
                    variant: "success"
                }));
            })
            .catch(error => {
                console.log(error);
                this.dispatchEvent(new ShowToastEvent({
                    title: "Error!",
                    message: "We could not update fd record, check error!",
                    variant: "Error"
                }));
            })
    }
}
